# Visão Geral das Tecnologias do Backend

Nesta página, você encontrará uma visão geral das principais tecnologias que você precisará dominar para contribuir com o desenvolvimento do backend da aplicação **Polvo**. Para aprender como configurar o ambiente, consulte a [página de configuração do ambiente](/config-ambiente).

<Tabs>
  <Tab title="Node.js">
    Node.js é uma plataforma robusta para executar JavaScript no servidor, permitindo a criação de aplicações escaláveis e de alta performance.
    
    <ul>
      <li><strong>Documentação:</strong> <a href="https://nodejs.org/en/docs/" target="_blank">Node.js Docs</a></li>
    </ul>
  </Tab>
  <Tab title="TypeScript">
    TypeScript adiciona tipagem estática ao JavaScript, ajudando a prevenir erros e a melhorar a manutenção do código.
    
    <ul>
      <li><strong>Documentação:</strong> <a href="https://www.typescriptlang.org/docs/" target="_blank">TypeScript Docs</a></li>
    </ul>
  </Tab>
  <Tab title="MongoDB">
    MongoDB é um banco de dados NoSQL orientado a documentos, ideal para armazenar dados de forma flexível e escalável.
    
    <ul>
      <li><strong>Documentação:</strong> <a href="https://docs.mongodb.com/" target="_blank">MongoDB Docs</a></li>
    </ul>
  </Tab>
  <Tab title="Mongoose" icon="node">
    Mongoose é uma biblioteca que facilita a modelagem e a validação dos dados no MongoDB através de esquemas, tornando as operações CRUD mais simples.
    <ul>
      <li><strong>Documentação:</strong> <a href="https://mongoosejs.com/docs/guide.html" target="_blank">Mongoose Docs</a></li>
    </ul>
  </Tab>
  <Tab title="Docker">
    Docker possibilita a criação e gerenciamento de containers, assegurando ambientes padronizados e isolados tanto para desenvolvimento quanto para produção.
    
    <ul>
      <li><strong>Documentação:</strong> <a href="https://docs.docker.com/" target="_blank">Docker Docs</a></li>
    </ul>
  </Tab>
</Tabs>
